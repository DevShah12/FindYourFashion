## Overview
FindYourFashion is a full-stack e-commerce web application built using PHP and MySQL. The project focuses on backend logic, relational database design, and structured server-side workflows to simulate a production-style online retail platform.

## Tech Stack
- PHP (Server-side development)
- MySQL (Relational database)
- HTML, CSS, Bootstrap (Frontend)
- Apache (Local environment)
- Git (Version control)

## Core Features
- CRUD operations for product management
- Categorized browsing (Men / Women / Winter Arrivals / Accessorie)
- Structured relational database schema
- Feedback submission system with server-side validation
- Custom error handling pages
- Modular backend architecture

## Architecture Focus
- Separation of frontend and backend logic
- Database-driven dynamic content rendering
- Maintainable and scalable backend structure

## How to Run Locally
1. Clone the repository
2. Import the MySQL database
3. Configure `config.php` with your local credentials
4. Run using Apache (XAMPP / WAMP)
5. Access via localhost
